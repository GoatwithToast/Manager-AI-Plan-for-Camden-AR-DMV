# Manager-AI-Plan-for-Camden-AR-DMV

This project aims to achieve key planning aspects of software engineering. This project is not affiliated with the actual Camden, AR DMV; the project is just a concept. 


# Project Summary

## From Description of Proposed Solution:

"Our team came up with this AI manager system to help out at a place like the DMV. It keeps tabs on how employees handle customers with license renewals, registrations, and all that, without the AI ever chatting with anyone directly. The goal is to give real managers solid numbers on speed, accuracy, and overall work so they can make fair calls on performance. We split the whole thing into hardware, software, processes, data, and the people side. It feels like a solid upgrade that still leaves room for human judgment."

## From Description of Request (in System Request):

"We need a manager AI that can monitor our employees to ensure they are on-task, and prevent mistakes. The manager AI will compare the information on clients that employees enter with that of which we have on the company’s database, and inform the employee and manager if there is a mismatch. The AI will tally up how many times mismatched data has been entered by each employee to flag them for investigation by the manager; the manager can remove “false-alarm” flags. The AI will also detect what software the employees are using and inform the manager if they are off-task (which will also be tallied). The AI will time the employees on tasks to calculate average completion times, and which employees are working in the most timely manner."
